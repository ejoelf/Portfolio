Proyectos

Rick and Morty

Sitio web inspirado en la serie "Rick y Morty", donde los usuarios pueden buscar personajes, ver sus datos en detalle, filtrar por género, ordenar por id y guardar sus favoritos.

  • React.js y Redux.js para el frontend.
  • Node.js y Express.js para el backend.
  • API externa para obtener los datos de los personajes.

Countries

Aplicación SPA que permite buscar y visualizar países del mundo, además de filtrar y ordenar los resultados. Incluye funcionalidades para crear actividades turísticas.

  • Frontend con React y Redux.
  • Backend con Node.js, Express y Sequelize.
  • Base de datos PostgreSQL con Sequelize.

Interfoods

Aplicación web Full Stack en equipo, que permite comprar alimentos de diferentes países, crear cuentas personales y dejar comentarios sobre platos específicos.

  • Autenticación de usuarios con Firebase.
  • Pasarela de pagos con Mercado Pago.
  • Envío de emails con Nodemailer.
